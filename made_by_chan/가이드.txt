아이템 필터 관련
블록
기본 구문은 각각의 조건을 지정하는 "표시" 및 "숨기기" 블록을 모아둔 것입니다. 표시 블록과 일치하는 경우 표시되고, 숨기기 블록과 일치하는 경우 숨겨집니다. 블록의 조건이 여러 개인 경우, 이 조건들이 블록과 일치하는 아이템에 모두 해당되어야 합니다.

이름
설명
Show
"표시" 블록 내의 조건과 일치하는 아이템이 있을 경우 해당 아이템이 표시됩니다.
예시
Show
                    Rarity >= Rare
Hide
"숨기기" 블록 내의 조건과 일치하는 아이템이 있을 경우 해당 아이템이 숨겨집니다. 일반 필터에서만 사용할 수 있습니다.
예시
Hide
                    BaseType "Scroll of Wisdom"
Minimal
일치하는 아이템의 라벨은 배경이 투명하게, 크기는 최소로 설정됩니다. 무자비 모드 필터에서만 사용할 수 있습니다.
예시
Minimal
                    BaseType "Scroll of Wisdom"
Continue
"표시" 또는 "숨기기", "최소화" 블록에서 "계속하기"를 사용하면 아이템이 이 블록과 일치할 때 해당 필더 규칙이 중단되지 않도록 할 수 있습니다. 예시를 통해, 드롭되는 모든 85레벨 이하의 아이템 중 홈이 6개인 아이템이나 80레벨 이상의 아이템인 경우를 제외하고는 모두 숨겨지는 것을 알 수 있습니다.
예시
Hide
                    ItemLevel < 85
                    Continue

                Show
                    ItemLevel > 80
                    Sockets 6
                    PlayEffect Blue
Import

예시
Import "MyCustomRules.filter"
Import "MyOptionalRules.filter" Optional

아이템 필터로 강조된 아이템 위에 색이 적용된 광선이 표시됩니다. 값이 임시(Temp)인 경우 광선은 아이템이 드롭되는 순간에만 나타나지만 그 외의 경우에는 계속해서 나타납니다.
값
이름	필요 조건	값 옵션
Colour		Red, Green, Blue, Brown, White, Yellow, Cyan, Grey, Orange, Pink, Purple
Temp		Temp
값 비활성화
None
예시
PlayEffect Red Temp
SetBackgroundColor
아이템 라벨의 배경 색상을 설정합니다.
값
이름	필요 조건	값 옵션
Red		0 - 255
Green		0 - 255
Blue		0 - 255
Alpha		0 - 255 (기본 값: 240)
예시
SetBackgroundColor 255 255 255 255
SetBorderColor
아이템 라벨의 테두리 색상을 설정합니다.
값
이름	필요 조건	값 옵션
Red		0 - 255
Green		0 - 255
Blue		0 - 255
Alpha		0 - 255 (기본 값: 255)
예시
SetBorderColor 255 0 0
SetFontSize
아이템 라벨의 글자 크기를 설정합니다.
값
이름	필요 조건	값 옵션
Size		1 - 45
예시
SetFontSize 30
SetTextColor
아이템 라벨의 텍스트 색상을 설정합니다. 무자비 필터에서는 알파 값이 80 이상이어야 합니다.
값
이름	필요 조건	값 옵션
Red		0 - 255
Green		0 - 255
Blue		0 - 255
Alpha		0 - 255 (기본 값: 255)
예시
SetTextColor 0 0 0
